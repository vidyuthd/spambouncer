Here Are Our Frequently Asked Questions

  1. How do we enable Spam Bouncer for filtering SMS?

    • Install Spam Bouncer app
    • Open Settings.app > Messages
    • Tap Unknown & Spam
    • Enable Spam Bouncer Extension

  2. How do we report SMS using Spam Bouncer

    • Open Messages.app
    • Open Settings.app > Messages
    • Tap Report all Messages
    • Select the message you want to report > Tap Report Button > Tap Done
    • Send the sms formatted for you in TRAI approved format to TRAI

  3. How to Register For DND using Spam Bouncer

    • Open Spam Bouncer app
    • Tap on Settings(Make sure you see Settings on Home Screen, if not refer to other question)
    • Tap Spam Reporting > Register for DND
    • Select yes on alert box to register for DND
    • Proceed and send the preformatted message to TRAI(1909)

  4. Why do I need to register for DND with TRAI?

    You can choose not to register for DND, its purely your own choice. Registering for DND will allow TRAI to send your preference to your telecom operator to avoid Unsolicited Commercial Communication (UCC)/ Telemarketing Calls / SMS. Once your DND preference is set, all telemarketers are required to obey your preferences within 7 days of their being set

  5. What is the difference between Spam Bouncer and other Spam Filtering Apps?

    Most other Spam filtering apps use keyword or rulebased filtering. This is very tedious because the user needs to configure keywords and rules. This is not a scalable process as the user has to keep on updating with new keywords/rules, this is where Machine Learning comes in, it simplies the process by automating the process of predicting whether an app is SPAM or not. Very few apps use machine learning and not all can guarantee Security & Privacy, as they would need to send the incoming message/text to their server to add to their collection to improve the accuracy of filtering. Spam Bouncer uses real word messages collected from people who willingly contributed their messages as sample set to train Machine Learning models. Spam Bouncer does not use data from the app in any way whether it is keywords added or preferences saved. Everything stays on device with the user. Spam Bouncer has an amazing and intutive User Interface for the user to get started off very easily and also has lots of functionality bundled in it, like reporting SPAM message, configuring or managing keywords, iCloud Sync and registering for DND.

  6. What is Deny list?

    This is a list of all words used for determining whether an SMS should be filtered or not. If the incoming message/text contains any word which is present in this list, it would be marked as SPAM always, regardless what Machine Learning prediction is. This gives user more control for super user.

  7. What is Allow List?

    This is the opposite of Deny list, a list of all words used for determining whether an SMS should be allowed or not. If the incoming message/text contains any word which is present in this list, it would not be marked as SPAM ever, regardless what Machine Learning prediction is. This gives user more control for super user.

  8. What does disabling a a word in the list do and how to do it?

    Disabling a word in the list(allow/deny) will make the filtering logic ignore that word. In other filtering logic behaves normally as if the word is not in the list and if the word is not present in the incoming message/text to be filtering using trained model for prediction (generated by Machine learning) would be applied for prediction purpose. For disabling a word in the list

    • Open Spam Bouncer app
    • Search the word or scroll to the word > Tap the word
    • Select the action "Disable the word" and confirm

  9. What does delete word from the list do?

    Delete word from list removes the word from the list so that app will continue to predict whether the incoming message is spam or not using normal mode which is by using model(trained by Machine Learning).

  10. I can't go to Settings screen in Spam Bouncer app

    If you see a lock icon, make sure to click Settings button and then I have enabled in settings button. Then you will screen a screen which matches the screen shown in website(Home section) which has Settings
